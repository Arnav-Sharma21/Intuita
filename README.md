# 🖤 Intuita

**Zero-Prompt AI Platform**

Intuita is a high-end, premium SaaS application designed to eliminate the need for user prompts. It provides a sophisticated "Zero-Prompt" UX using structured UI wizards, seamlessly integrating AI capabilities for non-technical users. 

Built with a stunning brutalist monochrome aesthetic and smooth GSAP animations, Intuita offers an Awwwards-worthy experience that redefines how users interact with AI.

## ✨ Features

- **Zero-Prompt Workspaces**: Specialized tools (like *EssayForge*) that require zero complex prompting.
- **Cinematic Visuals**: A premium brutalist UI with bento-grid layouts and dynamic micro-animations.
- **Live Output Streaming**: Real-time generative AI text streaming integrated directly into the workspace.
- **Modern Tech Stack**: Fully typed full-stack architecture with React, Express.js, and Vite.

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/Arnav-Sharma21/Intuita.git
   cd Intuita
   ```

2. Install dependencies
   ```bash
   npm install
   ```

3. Run the development server
   ```bash
   npm run dev
   ```

## 🛠 Tech Stack

- **Frontend:** React, TypeScript, Vite
- **Backend:** Express.js, Node.js
- **Animations:** GSAP (GreenSock)
- **AI Integration:** Anthropic API
